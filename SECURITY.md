# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please email security@athena-amm.com

Do not open public issues for security vulnerabilities.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Security Measures

- All smart contracts are audited
- AI model is trained on verified data
- No private keys stored on disk
