# Athena AI Model

Neural network for real-time sniper bot detection.

## Model Architecture

- Input: Transaction features (timing, amount, wallet history)
- Hidden layers: Pattern recognition
- Output: Risk score (0-1)

## Training Data

- Historical sniper transactions
- Known bot wallets
- MEV patterns
- Normal trader behavior

## Performance

- Inference time: <50ms
- Accuracy: Target 95%+
- False positive rate: <2%
