# Quick Start Guide

## Prerequisites
- Rust 1.75+
- Node.js 18+
- Python 3.10+
- Solana CLI

## Installation

1. Clone the repository
```bash
git clone https://github.com/rafealandrande/athena-amm.git
cd athena-amm
```

2. Run the build script
```bash
chmod +x scripts/build.sh
./scripts/build.sh
```

3. Deploy to devnet
```bash
chmod +x scripts/deploy.sh
./scripts/deploy.sh
```
